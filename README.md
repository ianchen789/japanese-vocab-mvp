# 日文單字學習 MVP

React、TypeScript 與 Vite 建立的前端專案。目前只完成 Phase 1 專案骨架。

## 啟動方式

需要 Node.js 20.19 以上版本。

```bash
npm install
npm run dev
```

依終端機顯示的本機網址開啟瀏覽器。

## 可用指令

```bash
npm run dev
npm run build
npm run lint
npm run preview
```

## 目前範圍

- 極簡首頁
- 新單字與複習的基礎畫面切換
- TypeScript mock data 型別
- 3 個範例單字與關聯資料

尚未實作新單字學習、小測驗、錯題補救、BAC 複習、SRS、API、登入或資料庫。
